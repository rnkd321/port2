창작 포트폴리오이며, 주제는 스팀 사이트를 좀 넓게 다시 만들어 봤습니다. 다 HTML로 만들고 CSS로
틀을 잡고 jquery로 작동하게 만들었습니다. 코드는 100% 다 본인이 짰으며 플러그인 쓴건 없습니다.
이 자작 사이트의 주요 기능을 설명해보겠습니다.
1. 맨 위에 해더 메뉴바가 있으며, jquery로 고정높이에 도달하면 fixed메뉴가 미끄러지듯이 나타나게 했습니다.
2. 왼쪽 aside엔 fixed한 세로 메뉴바를 구현했으며, 위치값은 CSS의 position으로 잡았으며 세련되 보이게
jquery로 easeInOutBack애니메이션을 넣었고 메뉴바 아이콘도 돌아가게 만들었습니다. 메뉴가 열리는 동시에
스크롤을 고정시키며 메뉴바 빼고 화면이 검은색으로 덮이게 만들었습니다.
3. 추천게임 슬라이더는 setInterval를 통해 자동으로 돌아가게 만들었고 마우스를 hover하면
clearInterval이 적용됩니다. 동시에 슬라이더 오른쪽에 간략소개창이 fadeIn으로 뜹니다. 간략소개창
안에는 자동으로 넘어가는 그림세트가 있으며, 메인슬라이더의 마우스 hover를 풀면 fadeOut 됩니다.
메인슬라이더의 오른쪽 그림을 호버하면 메인슬라이더가 같은 그림으로 바뀌게 만들었습니다. 슬라이더
양쪽은 좌우 버튼으로 다음밑쪽의 버튼은 슬라이더가 넘어가면 커지는 애니메이션이 동작하고 누르면 해당페이지로
넘어가게 만들었습니다.
4. 할인게임, 커뮤니티 추천게임 슬라이더는 setInterval이 없으며 모든 슬라이더는 태두리에 슬라더 박스를
만들고 박스에 position: relative값을 주고 슬라이더를 다 absolute로 박스안에 고정시켜 display: none에서
block으로 jquery의 addClass와 removeClass로 설정 한 것입니다.
5. 커뮤니티 추천게임 슬라이더는 처음엔 그림파일이 표시되게 하다가 마우스로 hover하면 그림이 display:none
이 되면서 뒤에 슬라이더가 자동 플레이되게 jquery로 구현 하였습니다.
6. 장르별 게임메뉴는 CSS로 버튼박스를 position: relative를 주고 밑에 5개의 페이지를 absolute로 겹쳐서
jquery로 몇 번째 버튼을 누르면 몇번째 페이지가 display: block으로 뜨게 해놓고 다른 페이지는 none상태로
안보이게 해놓았습니다.
